泛型接口：

public interface Inter<E> {  // element E    type T
  public abstract void show(E e);
}

//第一种明确方式：创建一个类，然后类实现一个接口的时候，明确泛型。
class Student implements Inter<String>{

  @Override
  public void show(String s) {
    System.out.println(s);
  }
}
// 第二种明确方式： 创建一个类 实现接口的时候， 把接口上的泛型 沿用下来，    等创建类的对象的时候明确    ArrayList
class Teacher<E> implements Inter<E>{

  @Override
  public void show(E e) {
    System.out.println(e);
  }
}

TreeSet (☆☆☆☆)

去重复 (把元素排了序，排序中遇到重复 去掉了重复)， 存取无序，没有索引

自然排序： 如果你想存储 自定义对象(学生对象)，因为TreeSet 有按照规则排序的特点， 那么你在自定义类中，必须提供对应的排序规则。 怎么提供呢？ 只需要让 自定义类 实现 Comparable 接口 重写里面的 compareTo方法， 并把规则写在方法里面就可以了。 自然排序这种方式相当于是： 我这个TreeSet容器 存储学生对象， 然而你学生对象 自己就具备 规则(compareTo),所以就给你排了。

案例1：

class Demo {
  public static void main(String[] args){
    TreeSet<Student> ts = new TreeSet<>();  
    ts.add(new Student(19));
    ts.add(new Student(78));
    ts.add(new Student(34));
    ts.add(new Student(97));

    System.out.println(ts); //  
  }
}
class Student implements Comparable<Student> {// Compare 比较意思  able 可以...的
  private int age;
  public Student(int age){
    this.age = age;
  }
  public String toString(){
    return "Student:"+age;
  }
  public int compareTo(Student s){
    return this.age - s.age
      /*
								返回值： 
									0 表示重复，表示当前添加的元素是重复的，不添加。
									正数 ： 往右添加
									负数：  往左添加
							*/
  }
}

比较器排序

比较器排序：如果你想用TreeSet存储 自定对象， 需要自己提供排序规则，如果这个自定对象自己不具备(自己没有实现Comparable接口) 那么就需要提供一个比较规则， 叫做比较器， Comparator ，里面有一个 compare 的方法 ，方法里面写规则。 我们用TreeSet这个容器，去存储学生， 然而学生并不具备比较规则， 提供第三方裁判。

/*
			TreeSet<Student> ts = new TreeSet<>();  
			ts.add(new Student(19)); // 运行出错。 因为 TreeSet存储学生对象 是要给学生对象排序的，但是现在是没有规则可以排序。
			*/
/*
			 TreeSet<Student> ts = new TreeSet<>(new Comparator<Student>() {
				@Override
				public int compare(Student o1, Student o2) { // o1 是即将添加的元素， o2 是已经添加过的元素
					return o1.getAge()- o2.getAge();
					//返回值：
						//0 重复 不添加当前元素
						//正数   往有去添加当前元素
						//负数   往左去添加 
				}
			});
			*/
TreeSet<Student> ts = new TreeSet<>((o1,o2)->o1.getAge()-o2.getAge());
ts.add(new Student(19));
ts.add(new Student(78));
ts.add(new Student(34));
ts.add(new Student(97));

System.out.println(ts); //[Student{19}, Student{34}, Student{78}, Student{97}]

两种排序方式的优先级： 如果TreeSet存储元素的时候， 元素自己也具备自然排序规则， 同时你也传入了比较器， 那么 TreeSet 底层会优先使用 比较器。

题目1： 有一个学生类， 有这些属性， 数学成绩 math ， 英语成绩 English ， 语文 Chinese 有5个学生， 按照学生的总成绩 从大到小排序。 如果总成绩相同， 请按照 数学排 如果总成绩同了 数学也同了，语文排。 语文也同了，安英语排。 英语也同了，按姓名排。 姓名也同了，就是同一个人 去除掉。

package Demo;

import java.util.TreeSet;

/*
有一个学生类， 有这些属性， 数学成绩 math ， 英语成绩 English ， 语文 Chinese
			有5个学生， 按照学生的总成绩 从大到小排序。
			如果总成绩相同， 请按照 数学排
			如果总成绩同了 数学也同了，语文排。
			语文也同了，安英语排。
			英语也同了，按姓名排。
			姓名也同了，就是同一个人 去除掉。
 */
public class Demo {
    public static void main(String[] args) {
        TreeSet<Stud> ts = new TreeSet<>();
        ts.add(new Stud("张三丰",98,99,100));
        ts.add(new Stud("风清扬",98,79,88));
        ts.add(new Stud("张无忌",97,99,100));
        ts.add(new Stud("张三丰",98,99,100));
        ts.add(new Stud("潘金莲",92,99,93));
        System.out.println(ts);
    }

}
class  Stud implements Comparable<Stud>{
    private String name;
    private int math;
    private int english;
    private int chinese;

    public Stud() {
    }

    public Stud(String name, int math, int english, int chinese) {
        this.name = name;
        this.math = math;
        this.english = english;
        this.chinese = chinese;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getMath() {
        return math;
    }

    public void setMath(int math) {
        this.math = math;
    }

    public int getEnglish() {
        return english;
    }

    public void setEnglish(int english) {
        this.english = english;
    }

    public int getChinese() {
        return chinese;
    }

    public void setChinese(int chinese) {
        this.chinese = chinese;
    }

    @Override
    public String toString() {
        return "Stud{" +
                "name='" + name + '\'' +
                ", math=" + math +
                ", english=" + english +
                ", chinese=" + chinese +
                '}';
    }

    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (o == null || getClass() != o.getClass()) return false;

        Stud stud = (Stud) o;

        if (math != stud.math) return false;
        if (english != stud.english) return false;
        if (chinese != stud.chinese) return false;
        return name != null ? name.equals(stud.name) : stud.name == null;
    }

    @Override
    public int hashCode() {
        int result = name != null ? name.hashCode() : 0;
        result = 31 * result + math;
        result = 31 * result + english;
        result = 31 * result + chinese;
        return result;
    }

    @Override
    public int compareTo(Stud o) {
        int sum = this.math+this.chinese+this.english;
        int sumo=o.chinese+o.english+o.math;
        int result =  sumo-sum ;
        int mathh=o.math-this.math;
        int chin =  o.chinese-this.chinese ;
        int eng = o.english- this.english ;
        int namecom =o.name.compareTo(this.name);
        if (result==0){
            result=mathh;
    }else if (result==0&&mathh==0){
            result=chin;
    }else if (result==0&&mathh==0&&chin==0){
            result=eng;
        }else if (result==0&&mathh==0&&chin==0&&eng==0){

         result=namecom;
        }


        return result;
    }
}
