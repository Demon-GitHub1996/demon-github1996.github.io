3.递归方式遍历每个文件夹和每个文件(必须掌握)

package Demo;

import java.io.File;
import java.io.IOException;

public class Demo2 {
  public static void main(String[] args) throws IOException {
    File file = new File("J:\\a");
    delete(file);

  }

  public static void delete(File f) {
    //遍历文件夹得到一个数组
    File[] file = f.listFiles();
    //判断是否为null&&file.lenth>0
    if (file != null && file.length > 0) {
      for (File f1 : file) {
        if (f1.isFile()) {
          f1.delete();
        } else {
          delete(f1);
        }

      }

    }
    f.delete();
  }
}

4.遍历文件夹获取文件类型和对应文件个数

package Demo;

import java.io.File;
import java.util.HashMap;
import java.util.Map;
import java.util.Set;

public class Demo3 {
  public static void main(String[] args) {
    HashMap<String, Integer> num = new HashMap<>();
    File file = new File("J:\\Develop");
    getNum(file,num);
    Set<Map.Entry<String, Integer>> entrySet = num.entrySet();
    entrySet.forEach(stringIntegerEntry -> System.out.println(stringIntegerEntry));
  }

  public static void getNum(File file, HashMap<String, Integer> num){
    File[] flies = file.listFiles();
    if(flies!=null&&flies.length>0){
      for(File f:flies){
        if(f.isFile()){
          String name= f.getName();
          String names= name.split("\\.")[1];//双转义字符！！！
          if(num.containsKey(names)){//如果names存在，值+1
            num.put(names,num.get(names)+1);
          }else {
            num.put(names,1);//不存在，初始化，值为1
          }
        }else {//不是文件，递归
          getNum(f,num);
        }
      }
    }
  }
}

I/O

1.i/o基本概念(重点)

什么是i/o

i -> input -> 读取数据 输入流
o -> output-> 写入数据 输出流

i/o分类

  • 按操作类型

    1. 输入流 FileInputStream

    2. 输出流 FileOutoutStream

  • 按数据类型

    1. 字节

      a1. 字节输入流

      a2. 字节输出流

    2. 字符

    b1. 字符输入流

    b2. 字符输出流

i/o应用场景

  1. 文本文件 优先使用字符流 字节流加编码集 2. 二进制文件 优先使用字节流
  2. 不明确类型 使用字节流(万能流)

2.输入流(读数据)

使用步骤(相关api尽量写全)

1.创建输入流
FileInputStream fis = new FileInputStream("path");
2.读取数据
fis.read(file对象);
3.关流
fis.close();

3.输出流(写数据)

使用步骤(相关api尽量写全)

1.创建输出流
FileOutputStream fos = new FileOutoutStream("path");
2.写数据
fos.write(file对象)
3.关流
fos.close();

4.文件复制(重点)

操作过程中不要一个个字节去读,效率太低

1.创建输入流输出流对象
FileInputStream fis = new FileInputStream("要读的文件");
FileOutputStream fos = FileOutputStream("要保存的文件");

2.文件复制
byte[] bytes = new byte[1024*8];
int length = 0;
while((length = fis.read(bytes))>0){//此处lenth是长度，若是内容，判定条件是！=-1
	fos.write(bytes,0,length);
}
