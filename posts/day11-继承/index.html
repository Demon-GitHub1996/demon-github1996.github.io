方法的重写：

子父类关系中， 子类出现了和父类方法声明 一模一样的这种方法， 就叫做方法的重写。 方法重写，只要和多态一起使用的时候， 才能体现出 代码的 拓展性和可维护性。

​ 面试题： 方法的重写和重载有什么区别 重载： 同一个类中 方法名相同 参数列表必须不同 返回类型无关 重写： 子父类中 子类出现了和父类方法声明 一模一样的方法

方法重写的注意事项：

1：父类私有的方法， 子类不能重写？？

// 重写的目的 很大80%以上的情况 都是结合着多态去使用， 如果你不算重写， 多态用不了。

class Person { 
private void show(){ 
System.out.println("haha");
}
}
class Student extends Person {
public void show(){  // 不会报错   但是 这不是重写。 多态用不了的。
System.out.println("show");
}
}

​ 2：静态的方法不算重写

class Person { 
public static void show(){ 
System.out.println("haha");
}
}
class Student extends Person {
public static void show(){  // 不会报错   但是 这不是重写。 多态用不了的。
System.out.println("show");
}
}

3: 子类重写父类的方法 权限要大于等于父类权限

class Person { 
public void show(){ 
System.out.println("haha");
}
}
class Student extends Person {
protected  void show(){ //编译报错protected < public

}

构造方法：

子类在创建对象的时候，必须去调用父类的构造方法（语法规定），为什么语法这个规定呢？ 是因为：子类创建对象要使用的时候，保不准就会用到父类的内容， 所以 一定是先去把父类在内存中初始化，那么如何把父类在内存中初始化么， 就是访问父类的构造方法，这就是在内存中初始化。 父类初始化完了之后，子类再继续初始化子类的。

所以 构造方法的访问特点有如下几个：

1: 子类的所有构造方法的第一行 都默认有一个 super();（只不过你看不到而已） 用于访问父类的无参构造。 2： 而且当你显示给出 super(), super(参数)，， 系统就不再默认提供 super() 3： 访问构造方法的语句 必须放在 构造方法的第一行。 4： 当父类没有无参构造方法的时候， 你必须要做 子类初始化之前先初始化 父类这一点， 访问 父类的其他有参构造。 访问 子类其他的构造方法。 无论哪一种， 你只需要做到 子类初始化之前 必须先初始化父类就可以。 5： 子类初始化之前，必须先初始化父类，初始化几次 一次（必须一次），所以 this() 和super() 都必须放在第一行，所以他俩不能共存。

**以上细节：你只需要记住原理，就不会忘。 **

==子类初始化之前必须先初始化父类，而且必须是初始化一次。==

抽象类(☆☆☆☆)：

==**概念： 如果一个类中有抽象方法， 那么此时这个类 就必须是抽象类 **==

【因为抽象类是不能创建对象的， 如果能创建对象就可以调用 抽象方法了 然而抽象方法没有方法体 没法调用，刚才也给你看到了 抽象方法的作用是为了给子类提供一个方法声明的模板的】

abstract class BaseStudentController {

1：抽象类不能创建对象

疑问：抽象类创建不了对象 是不是因为他没有构造方法啊， 当时学构造方法的时候 ，要想创建对象就必须要执行他的构造方法， 然而此时他却创建不了对象，那岂不是就意味着他没有构造方法嘛 。 答案并不是的， 抽象类里面是有构造方法的， 但是他的构造方法并不是为了让你创建对象使用的， 而是为了让子类继承父类 子类创建对象的时候 访问的。 抽象类不能创建对象 是因为 abstract 这个关键字的作用。

final(☆☆☆☆☆) ：“最终"的意思

​ 这是一个修饰符： 可以修饰什么呢？

修饰类： 被final的类 最终类， 意思是不能被继承。

final class Student {
}
class LittleStudent extends Student{ // 编译报错

}
修饰方法： 被final修饰的方法  最终方法， 意思不能被重写了。
public class Student {
public final void study(){  

}
}
class LittleStudent extends Student{
public void study(){  // 编译报错。

}
}

修饰变量： 被final修饰的变量， 最终变量。 不能被重新赋值了。

常量： 自面值常量：

int a = 10;
a = 20;
System.out.println(a); //20

10 // 就是自面值常量 
20 // 字面值常量。
3.1415926 // 有的时候 这样的数据 确实是常量 自面值常量  但是 很不好记。
// 能不能给这个常量 起个名字呢。
// double a = 3.1415926 ;   
// a = 5.4;
自定义常量：
final double a = 3.1415926 ;
// 那么想在这个 a 这个名字 就是 自定义常量。

1：不能被再次赋值。

public class Student {
